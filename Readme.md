'Odin Recipes' project is the application of all the Foundation course's HTML lessons which I completed in [The Odin Project](https://www.theodinproject.com/paths/foundations/courses/foundations).

In this project I had to create a basic recipe website of your 3 favorite dishes. This website consists of a main index page which will have links to those recipes. 

I have added recipes of my 3 favorite dishes:- Pav Bhaji, Panipuri & Bhel Puri.
